🔍 Metodología

Limpieza de datos — Se identificaron y removieron registros duplicados, se trataron valores nulos, salarios negativos, se realizo la union de los datos y la creacion de un nuevo DataFrame con los datos ya limpios para su posterior trabajo.
Análisis descriptivo — Revisión de variables numéricas y categóricas, detección de valores atípicos y estudio de correlaciones entre variables.
Visualización de datos — Construcción de gráficas para observar distribuciones, relaciones entre variables y comparaciones por segmento de cliente.
Evaluación — Se desarrollo todas las dudas planeadas para el proyecto, creando graficas y la descripcion de ellas.

📊 ¿Que nos dicen los datos? (conclusiones)

- Estacionalidad marcada: las reservas aumentan notablemente en verano (junio–agosto) y vuelven a repuntar en diciembre, en línea con las temporadas vacacionales.

- Educación sin efecto sobre los vuelos reservados: los promedios son prácticamente iguales entre todos los niveles educativos, por lo que esta variable no parece explicar diferencias en el comportamiento de vuelo.

-Los clientes con tarjeta Aurora, aunque son menos numerosos, tienen una relación económica más valiosa con la empresa y aportan mayor beneficio individual.
